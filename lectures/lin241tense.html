Preliminaries

A problem

  • Consider the following:

    (1) John is reading Doctor Zhivago.

    (2) John read Doctor Zhivago.

    (3) John will read Doctor Zhivago.

  • What are the truth conditions of (1), (2), and (3)?

    How are they distinct from each other?

    …And how are they related to each other?

Informally…

  • (1)-(3) have the same core meaning

  • They differ in tense

  • If "John read Dr Zhivago " is true at some point in time, then "John is reading Dr Zhivago" is true at some earlier point in time

  • If "John will read Dr Zhivago ."is true at some point in time, then "John is reading Dr Zhivago" is true at some later point in time

Our current system

  • Currently, we just ignore tense

  • Set theory has no tense.

  • ⟦John is reading Dr Zhivago. ⟧M,c,g,w=1
    iff ⟨J′,DrZh′⟩ ∈ V(is reading)

  • ⟦John read Doctor Zhivago. ⟧M,c,g,w= 1
    iff ⟨J′,DrZh′⟩ ∈ V(readpst)

  • ⟦John will read Doctor Zhivago. ⟧M,c,g,w= 1
    iff ⟨J′,DrZh′⟩ ∈ V(will read)

Our current system

  • is reading, readpst, and will read are distinct relations in our model.

  • We could stipulate the entailments that hold between them …

  • … But then we would have to do so for every verb.

  • We want to be able to derive the similarities and differences between tenses systematically.

Towards a tense system

(1) John is reading Doctor Zhivago.

(2) John read Doctor Zhivago.

(3) John will read Doctor Zhivago.

  • Back to our informal intuitions:

    • (1)-(3) have the same core meaning

    • They differ in tense

  • Now let's formalize our intuitions

Towards a tense system

Let's separate tense from the core meaning

  • ⟦John is reading Dr Zhivago. ⟧M,c,g,w=
    ⟦pres⟧M,c,g,w(⟦ John read Dr Zhivago.⟧ M,c,g,w)

  • ⟦John read Dr Zhivago. ⟧M,c,g,w=
    ⟦pst⟧M,c,g,w (⟦John read Dr Zhivago.⟧ M,c,g,w)

  • ⟦John will read Dr Zhivago.⟧M,c,g,w=
    ⟦fut⟧M,c,g,w (⟦John read Dr Zhivago. ⟧M,c,g,w )

A tensed sentences separable into tense and a sentence radical

The LF of a tensed sentence

How do we interpret Stense?

  • The truth of a tensed sentence is evaluated at the time of utterence (TU)

  • The state of affairs encoded in the sentence radical is interpreted relative to TU

    • The truth of "John will read Dr Zhivago" depends the state of affairs at some point in time that comes after the time it is uttered

  • How do we encode that?

Introducing events

  • In our current system, verbs denote relations between individuals.

    • V(read) = {⟨x, y⟩ : x read y}

  • Sentence radicals, then, denote propositions.

  • Our intuitive sense is that they describe "states of affairs"

  • Semanticists call these "states of affairs" Events.

  • Events are a new type of semantic object in our model

Adding Events to our Model

  • Our current conception of our model: M = ⟨D, V⟩

  • The domain of discourse D is actually split into subdomains:

    • Domain of entities De
      {John', Dr Zhivago', …}

    • Domain of truth-values Dt
      {0,1}

  • Now we add the domain of events Dv
